

# AI Appointment System

An AI-powered appointment booking and management system built using ASP.NET Core MVC.

## Features


- Appointment Booking
- Admin Dashboard
- AI Chatbot Assistant
- Responsive UI Design
- Appointment Management
- Clean Architecture

## Technologies Used

- ASP.NET Core MVC
- C#
- SQL Server
- Entity Framework Core
- Bootstrap
- JavaScript
- HTML/CSS

## Installation

1. Clone repository

```bash
git clone https://github.com/YOUR_USERNAME/ai-appointment-system.git
```

2. Open project in Visual Studio

3. Restore packages

4. Update database connection string

5. Run project
